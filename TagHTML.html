<!DOCTYPE> Tag untuk menentukan tipe dokumen
<html> Tag untuk membuat sebuah dokumen HTML
<title> Tag untuk membuat judul dari sebuah halaman
<body> Tag untuk membuat tubuh dari sebuah halaman
<h1> - <h6> Tag untuk membuat heading
<p> Tag untuk membuat paragraf /td>
<br> Memasukan satu baris putus
<hr> Tag untuk membuat perubahan dasar kata didalam isi
<!--....--> Tag untuk membuat komentar
<abbr> Tag untuk membuat sebuah singkatan
<address> Tag untuk membuat kontak alamat
<b> Tag untuk membuat huruf bercetak tebal
<bdi> Mengisolasi bagian dari teks yang dapat diformat dalam arah yang berbeda dari teks lain di luarnya
<bdo> Mengganti arah teks
<cite> Tag untuk membuat judul karya
<code> Tag utuk membuta potongan code komputer di antara text
<del> Tag untuk membuat teks yang telah dihapus dari dokumen
<dfn> Tag untuk membuat sebuah istilah definisi
<i> Tag untuk membuat tulisan miring (italian)
<ins> Tag untuk membuat teks yang telah dimasukkan ke dalam dokumen
<kbd> Tag untuk membuat input keyboard
<pre> Tag untuk membuat teks berformat
<q> Tag untuk membuat kutipan pendek
<s> Tag untuk membuat teks yang tidak lagi benar
<samp> Tag untuk membuat contoh keluaran dari komputer
<small> Tag untuk membuat teks kecil
<strong> Tag untuk membuat teks penting
<sub> Tag untuk membuat teks subskrip (pagkat di bawah)
<sup> Tag untuk membuat teks superscripted (pangkat di atas)
<time> Tag untuk membuat tanggal / waktu
<u> Tag untuk membuat garis bawah (underlline)
<var> Tag untuk membuat sebuah variabel
<wbr> Tag untuk membuat kemungkinan garis putus
<form> Tag untuk membuat sebuah form HTML untuk input pengguna
<input> Tag untuk membuat sebuah kontrol input
<textarea> Tag untuk membuat sebuah kontrol input multibaris (text area)
<button> Tag untuk membuat sebuah tombol yang dapat di klik
<select> Tag untuk membuat sebuah daftar dop-down
<optgroup> Tag untuk membuat sebuah kelompok pilihan yang terkait dalam daftar drop-down
<option> Tag untuk membuat pilihan dalam daftar drop-down
<label> Tag untuk membuat sebuah label untuk sebuah elemen <input>
<fieldset> Grup unsur terkait dalam bentuk
<legend> Tag untuk membuat sebuah caption <fieldset>, <figure> atau <details>
<output> Tag untuk membuat hasil penghitungan
<frame> Tag untuk membuat sebuah widow (bingkai) dalam sebuah frameset (tidak disupport lagi di HTML5)
<frameset> Tag untuk membuat satu set bingkai (tidak di support lagi di HTML5)
<noframes> Tag untuk membuat sebuah konten alternatif untuk pengguna yang tidak mendukung frame (tidak disupport lagi di HTML5)
<iframe> Tag untuk membuat sebuah bingkai
<img> Tag untuk membuat gambar
<map> Tag untuk membuat gambar peta
<area> Tag untuk membuat area dalam gambar peta
<canvas> Digunakan untuk menggambar grafik, melalui scripting
<figcaption> Tag untuk membuat sebuah caption untuk elemen <figure>
<figure> Tag untuk membuat konten mandiri
<audio> Tag untuk membuat isi suara (file mp3)
<source> Tag untuk membuat sumber beberapa media untuk elemen media( <video> dan <audio>)
<track> Tag untuk membuat trek teks untuk elemen media (<video> dan <audio>)
<video> Tag untuk membuat sebuah video atau film (file mp4)
<a> Tag untuk membuat hyperlink
<link> Tag untuk membuat hubungan antara dokumen dan sumber daya eksternal (paling sering digunakan untuk style link ke style sheet)
<nav> Tag untuk membuat tag untuk membuat navigasi link
<ul> Tag untuk membuat daftar dengan selain nomor contohnya dengan dots/titik
<ol> Tag untuk membuat daftar dengan nomor
<li> Tag untuk membuat sebuah item daftar
<dir> Tag untuk membuat sebuah daftar direktori
<dl> Tag untuk membuat sebuah daftar definisi (tidak disupport lagi di HTML5)
<dt> Tag untuk membuat istilah (item) dalam daftar definisi
<dd> Mendefinisikan deskripsi dari sebuah item pada daftar definisi
<menu> Tag untuk membuat deskripsi dari item dalam daftar definisi
<command> Tag untuk membuat sebuah tombol perintah bahwa seorang pengguna dapat meminta
<table> Tag untuk membuat tabel
<th> Tag untuk membuat sebuah sel header table
<tr> Tag untuk membuat baris dalam sebuah table
<td> Tag untuk membuat sel dalam sebuah tabel
<thead> Mengelompokkan isi hearder dalam sebuah tabel
<tbody> Mengelompokkan isi tubuh dalam sebuah tabel
<tfoot> Mengelompokka isi footer dalam sebuah table
<col> Menentukan properti kolom untuk setiap kolom dalam elemen <colgroup>
<colgroup> Menentukan kelompok dari satu atau lebih kolom dalam sebuah tabel untuk diformat
<style> Tag untuk membuat informasi style
<div> Tag untuk membuat sebuah bagian dalam dokumen
<span> Tag untuk membuat sebuah bagian dalam dokumen
<header> Tag untuk membuat sebuah header untuk dokumen atau bagian
